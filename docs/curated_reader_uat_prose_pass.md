## Reader Promise

This pass should let a human reader understand why tribunal review is useful
only when it is bounded by a dossier, separates reviewer roles, preserves
dissent, blocks review laundering, and changes the state of the ledger or
execution system.

The reader throughline is:

> review must change state.
