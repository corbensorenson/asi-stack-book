## Reader Promise

A human reader should leave this chapter understanding that failure analysis in
the ASI Stack is not a dramatic risk catalog. It is an obligation map: each
failure mode names the boundary that should detect it, contain it, preserve
evidence about it, or block promotion until the residual is handled.
