# v1.0 Evidence Transition Pilot

This folder contains the first v1.0 evidence-transition pilot records. The current pilot has twenty accepted no-change records.

The pilot is intentionally conservative: each record documents a no-change decision for a narrow book-method or architecture-control claim. The records show that repository artifacts, validators, synthetic harnesses, and finite-record proofs exist, while also recording why those artifacts do not yet justify moving the corresponding chapter core claim above `argument`.
