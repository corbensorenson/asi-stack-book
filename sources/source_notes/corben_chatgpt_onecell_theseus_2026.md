## Thesis

A compact, weight-tied recurrent candidate could sit behind a stable Cognitive
Kernel ABI while Theseus governs architecture tournaments, promotion, rollback,
and retirement across heterogeneous kernels.

## Mechanisms

- One shared transition cell with typed state lanes.
- Inner latent recurrence plus outer exact branch-preserving search.
- Question routing as value-of-computation control.
- Verified abstraction learning and a frozen-core ratchet.
- Theseus outside the kernel as independent lifecycle authority.
- Separate observation, external world, belief, prediction, goal, permission,
  exact memory, heuristic memory, and residual state.
- Use typed query and program IRs so deterministic retrieval or execution cannot
  hide inside an opaque candidate generator.
- Run reflex, reaction, planning, reflection, and consolidation lanes at distinct
  cadences while testing interference from shared weights.
- Restrict neural test-time memory to attributable, revocable heuristic state;
  keep exact identity, effects, provenance, and receipts outside it.
- Allow core updates only during quiescent stabilization with replay, regression,
  rollback, and descendant invalidation.

## Evidence

The conversation makes OneCell falsifiable and explicitly unproven. No code,

## Failure Modes

- Self-evaluation or architecture self-promotion.
- Hidden complexity in language adapters, retrieval, tools, or outer search.
- State-lane interference and failure to extrapolate recurrence depth.
- Wrong-attractor convergence when recurrence substitutes for explicit search.
- Syntactic trace compression mistaken for behaviorally equivalent abstraction.
- A semantic graph mistaken for reality rather than a fallible belief representation.

## Claims To Add Or Update

- Define a substrate-neutral ABI and architecture-RSI lifecycle.
- Test reflex, reaction, deliberation, and specialist routes above that ABI.
- Preserve OneCell as one tournament lane with explicit defeat conditions.

## Open Questions

- Can a shared cell support heterogeneous state without destructive interference?
- Can an independent evaluator distinguish real abstraction from hidden assistance?
- Does world-model state improve intervention prediction and planning over
  model-free search under matched cost?
- Does useful recursion deepen beyond trained regimes, or does performance arrive
  in the first few steps while nominal depth adds cost?
