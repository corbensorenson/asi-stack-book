Jupyter Book is an external comparator for executable and computational books. It helps position the ASI Stack living book against a known pattern where notebooks or Markdown, executable code, cross references, and web publication are combined into book-like artifacts.

## Mechanisms

- Author with notebooks or Markdown.
- Execute code and publish computational content.
- Cross-reference content across a book-like site.
- Publish to web-facing book artifacts.

## Evidence

- The official documentation presents Jupyter Book as a tool for computational books and web publication.
- This repository has not built a Jupyter Book, imported Jupyter Book execution, or validated this project through the Jupyter Book toolchain.
