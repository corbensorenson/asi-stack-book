# Source Inbox

Use this folder as a local staging convention for new source material.

Recommended flow:

1. Add a source record to `sources/source_inventory.json`.
2. Place exported text or notes here only when publication and storage are allowed.
3. Ask Codex to triage the source against `book_structure.json`.
4. Move durable notes to `sources/source_notes/<source-id>.md`.
5. Keep private or restricted source text out of the public repo unless explicitly approved.
