# Circle Cyclic Memory Receipt Slice

This directory records a public-safe ASI-side import of one local Circle
Calculus cyclic-memory receipt run. The slice is structural evidence only:
residue, winding, alias-load, theorem IDs, receipt fingerprints, command-output
digests, and non-claims.

It does not vendor Circle, rerun Circle from CI, prove learned memory behavior,
or create a support-state transition.
