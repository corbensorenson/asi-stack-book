# Theseus Public Task-Bundle Import

This directory contains a public-safe ASI Stack import fixture for one Project
Theseus public task-bundle result. It records report hashes, summary counts,
public-boundary flags, gate counts, residual counts, artifact gaps, and
non-claims only.

It does not copy public prompts, tests, solutions, traces, scores, or candidate
code. It does not claim a clean live Project Theseus replay or promote any ASI
Stack chapter core claim.

Run:

```bash
python3 scripts/validate_theseus_public_task_bundle_import.py
```
