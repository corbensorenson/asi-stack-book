# Theseus Governance-Rights Receipt Suite Import

This experiment records a sanitized public-safe summary of the local Project
Theseus governance-rights receipt suite. It does not copy the raw Theseus
report, private payloads, private path fields, prompts, tests, solutions,
candidate traces, score labels, checkpoints, or training rows into this public
repository.

Run:

```bash
python3 scripts/validate_theseus_governance_rights_receipt_suite_import.py
```

To regenerate the tracked result after intentional fixture changes:

```bash
python3 scripts/validate_theseus_governance_rights_receipt_suite_import.py --write-result
```
