# Preserved prevalidation pilot

This directory preserves the first 16 generated candidates and their initial
result. It is not evidence for a book claim. The pilot exposed two protocol
defects before flagship validation: every generation omitted the requested
explicit plan line, and route effects used in-memory repository snapshots
rather than fresh Git worktrees.

The versioned amendment at
`experiments/post_v2_evidence_program/amendments/governed_work_v1.json` freezes
the corrected plan-to-code and real-worktree protocol while retaining the
same task corpus, seeds, pinned model, attacks, metrics, and allowed outcomes.
