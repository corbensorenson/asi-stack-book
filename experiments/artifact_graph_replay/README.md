# Artifact Graph Replay Harness

This experiment directory contains synthetic artifact-graph replay fixtures used
by `python3 scripts/validate_artifact_graph_replay.py`.

The harness checks cross-record consistency between an artifact graph record,
typed job, context transaction, semantic certificate, and replay attempt. It is
record-discipline evidence only. It does not prove deployed artifact-graph
behavior, real replay execution, audit reconstruction quality, source
interpretation, or chapter support-state promotion.
