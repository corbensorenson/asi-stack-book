# Resource Workflow Trace Fixtures

These fixtures support:

```bash
python3 scripts/validate_resource_workflow_trace.py
```

They are deterministic public-safe workflow traces for Resource Economics. They
check route selection, scheduler ordering, displaced-cost residuals, protected
overhead, quality predicates, and bounded simulation-transfer language.

They do not run a deployed scheduler, measure model quality, prove economic
outcomes, validate physical feasibility, reproduce TokenMana or PlanForge, or
promote any support state.
