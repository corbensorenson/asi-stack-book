# Reader Key-Figure HTML Probe

This directory stores a public-safe result record for the rendered curated
reader HTML key-figure probe.

The probe builds the tracked curated reader manuscript in a temporary
workspace, renders it to HTML, and checks the ten draft key-figure blocks in
the rendered HTML DOM for image references, copied SVG assets, alt text,
captions, responsive image classes, and visible non-claim boundary paragraphs.

It is not a release approval, not final figure-artifact review, not a visual
design review, and not evidence for any chapter core claim.
