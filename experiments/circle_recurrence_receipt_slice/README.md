# Circle Recurrence Receipt Slice

This directory records a public-safe ASI-side import of one local Circle
Calculus recurrence-schedule receipt run. The slice is structural evidence
only: finite loop-period fields, active/inactive token work traces, scheduled
work-saving accounting, whole-period shift invariants, theorem IDs, receipt
fingerprints, command-output digests, planner recommendations, and non-claims.

It does not vendor Circle, rerun Circle from CI, prove deployed recurrence
behavior, prove reasoning quality, prove memory savings, prove long-context
quality, or create a support-state transition.
