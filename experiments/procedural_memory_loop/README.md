# Procedural Memory Loop Harness

This experiment directory contains synthetic procedural-memory loop fixtures
validated by `python3 scripts/validate_procedural_memory_loop.py`.

The harness checks record discipline for trace clustering, negative-example
preservation, abstraction, regression routing, and retirement triggers. It does
not prove deployed loop detection, tool synthesis, regression quality, routing
quality, monitoring automation, source interpretation, or support-state
promotion.
