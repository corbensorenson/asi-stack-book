# Routing Decision Lease Harness

This experiment directory contains synthetic routing-head fixtures validated by
`python3 scripts/validate_routing_decision_lease.py`.

The harness checks record discipline for least-capable adequate route
selection, overprivileged specialist rejection, authority grants that remain
inside the selected specialist's registry envelope, missing-readiness fallback,
expired lease residualization, rejected-candidate evidence, residual ownership,
and MoECOT source-boundary non-claims. It does not prove routing accuracy,
learned-router quality, deployed authority enforcement, MoECOT runtime replay,
specialist quality, cost-quality optimization, benchmark performance, or
support-state promotion.
