# Theseus Report Bundle Audit

This directory contains a public-safe repository fixture for the Project Theseus
report-bundle audit. It checks whether a bundle-shaped record carries the
minimum contract, work-board, gate, residual, replay, artifact-gap,
intervention-ladder, publication-boundary, and non-claim fields before the book
treats it as implementation-reference evidence.

The audit is intentionally narrow. It does not import private Project Theseus
payloads, replay Project Theseus, reproduce benchmarks, inspect a live
dashboard, approve public task bundles, or promote any chapter support state.
