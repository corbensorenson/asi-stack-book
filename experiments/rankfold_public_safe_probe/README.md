# RankFold Public-Safe Replay Probe

This directory stores the result record for a fresh local RankFold replay over
a generated public-safe synthetic text fixture.

The result records command outcomes, hashes, ratios, a roundtrip-exact unpack
check, and a corrupt-archive negative control. It does not store input bytes,
archive bytes, datasets, private paths, private keys, or unpublished
implementation content.

This is not a compression benchmark, not a NeuralFold result, not a codec
correctness proof, not a downstream utility test, and not a chapter-core
support-state promotion.
