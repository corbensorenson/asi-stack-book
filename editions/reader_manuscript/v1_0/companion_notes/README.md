# Reader Companion Notes

This directory holds drafting companion notes for dense reader, e-reader, and
audio treatment. These files are review support only. They are not reader
release records, not artifact reviews, not support-state transitions, and not
equal authority beside the live AI/research book.

Meaning-critical limits must remain in the reader spine. A companion note may
explain vocabulary, summarize dense record fields, or guide audio treatment,
but it must not become the only place where a proof boundary, governance
boundary, release blocker, or non-claim appears.
